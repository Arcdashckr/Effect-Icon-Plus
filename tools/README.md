# 🛠️ Effect-Icon-Plus Developer ToolKit

This folder is a fully-fledged, Python-powered automation studio designed to add new mod compatibility to the **Effect-Icon-Plus** resource pack, manage color palettes, and automatically update repository displays and documentation.

All processes are interconnected via Python scripts leveraging the ImageMagick library for optimal workflow efficiency.

---

## 🚀 Quick Start

To manage the entire toolkit from a single dashboard, simply double-click and run **`start.bat`** inside the folder. You will be greeted by the following integrated menu:

- [1] Extract and Scale Mod Icons
- [2] Frame and Merge Based on Color Palette
- [3] Generate Preview Display Grid
- [4] Exit

---

## 📦 Step-by-Step Workflow & File Responsibilities

### 1️⃣ Step 1: Icon Extraction (extract_and_scale.py)
* **Purpose:** Automatically scans and detects effect icons inside `.jar` or `.zip` mod files.
* **How it works:** Reads the mod archive, captures all 16x16 pixel-art icons inside the `assets/<mod_id>/textures/mob_effect/` directory, scales them crisply (Nearest Neighbor) to **54x54**, and saves them under `tools/extracted/<mod_id>/`.

### 2️⃣ Step 2: Color Analysis & Framing (merge_frames.py)
* **Purpose:** Extracts the color palette from raw icons, populates `colors.json`, and merges the 9-stage colored frame slices (`frames/`) onto the icon.
* **Advanced Features:**
  * **Speed Filter:** Prompted on run. If an output icon already exists in the repository and the JSON remains unchanged, it skips processing—saving immense amounts of time.
  * **Contrast Booster:** Prevents ImageMagick's auto-generated colors from turning out dull or greyish by mathematically expanding the gradient range to make the borders pop.
  * **Dual-Block Management (colors.json):**
    * "auto": The untouched reference block generated by ImageMagick, mapped directly to the 9 border frames.
    * "manual": Reserved for your custom hex entries. Additionally, an `_all` key appended to the effect name stores all unique raw colors found in the icon for easy reference (does not affect rendering).

### 3️⃣ Step 3: Display & Documentation Automation (create_display.py)
* **Purpose:** Arranges the completed 54x54 framed icons into a 10-column layout preview sheet and automatically regenerates repository documentation.
* **Options:**
  1. *Vanilla Only:* Scrapes `assets/minecraft` and outputs `effect_display_vanilla.png`.
  2. *Specific Mod:* Generates a layout exclusively for a single manually typed mod folder.
  3. *All Mods Separately:* Generates distinct sheets for each independent mod folder.
  4. *Merge Everything:* Unifies Vanilla and mods into one giant `effect_display_mod.png` sheet to guarantee your raw link assets (`README.md` etc.) never break.
* **Advanced Features:**
  * **Lossless Dynamic Scaling:** Instead of forcing fixed dimensions that crop the image, the script allows vertical length to dynamically stretch based on icon count, crisp-scaling pixels by 400% (`_upscaled.png`) for a flawless GitHub showcase.
  * **Live README Integration:** Injects live stats directly into your root `README.md` via sleek Modrinth-themed badges, automatically ticking up the total mod and icon counters.
  * **Automated `compatibility.md`:** Scrapes supported mods and icon counts to compile an absolute, alphabetized Markdown compatibility table effortlessly.

## 🛠️ Requirements

The following dependencies must be installed on your system and appended to your Environment Variables (PATH):
1. **Python 3.x**
2. **ImageMagick** (The global `magick` command must be executable via terminal.)

## 📁 Repository Structure Reference

- tools/
  - extracted/           # 54x54 raw mod icons from Step 1
  - frames/              # White border frame slices from 1.png to 9.png
  - start.bat            # Main control panel script
  - colors.json          # Color database (auto/manual)
  - extract_and_scale.py # Mod extractor script
  - merge_frames.py      # Smart color-mapping and frame compiler script
  - create_display.py    # Display, README, and table updater script
